 Database >> 
 A database is an organized collection of data that is stored electronically so it can be easily accessed, managed, and updated.

 DBMS >>
 A DBMS (Database Management System) is software that allows users to create, store, retrieve, update, and manage data in a database.

 Types Of Database >>

 1. Relational [ RDBMS ]
 - data stored in table 
 - we use sql to work with rdbms 
 - e.g. MySQL , oracle , postrageSQL etc 

 2. Non - Relational [ NoSQL]
 - data not stored in tables 
 - e.g. mongoDB

 What is SQL >> 
 - SQL (Structured Query Language) is the standard language used to     CRUD create, retrieve, update, and delete data in a Relational Database Management System (RDBMS).
 - SEQUEL = Structured English Query Language , made by IBM

 Table >> 
 A table is a structured way of storing data in a database. It organizes data into rows and columns, similar to an Excel sheet.

 Schema >>
 A schema is the structure or blueprint of a database. It defines how data is organized, including tables, columns, relationships, and constraints i.e it tells us what tables exist, what columns they have, and how data is related.
